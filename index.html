--- layout: home search_exclude: true image: images/IM1.png --- ![]({{site.baseurl}}/images/portada.png) Este proyecto explora un dataset de datos referentes a partidos de futbol pertenecientes a Octosport and Sportmonksde. El objetivo principal es entender como variables referentes a partidos anteriores afectan resultados futuros. En última instancia, y sin éxitos mayores, se busca modelar dicha interdependencia con modelos de machine learning. # Bloques en los que se organiza el proyecto
